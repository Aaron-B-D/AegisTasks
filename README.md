# Nombre
AegisTasks

# Descripción
AegisTasks es una aplicación de escritorio desarrollada en .NET WinForms para la gestión y ejecución de tareas automatizadas sobre bases de datos SQL Server.
Permite definir tareas con dependencias, controlar su ejecución y registrar auditorías, garantizando seguridad y trazabilidad en entornos con alta concurrencia.

# Funcionalidades
  -CRUD de tareas y plantillas.
  -Ejecución secuencial o paralela.
  -Control de bloqueos y recursos.
  -Monitorización y registro de auditoría.

# Tecnologías
  -C# (.NET 6)
  -Windows Forms
  -SQL Server
